APIs

Lifecycle Stage

APIs have the following lifecycle stages:

released

the API is released and can be used in API Products

deprecated

the API is deprecated and cannot be added to API Products. Existing API Products are unaffected

Managing APIs

View API

  • user is member of the business group with roles API Team

Create API

  • user is member of the business group with roles API Team

Update API

  • user is member of the business group with roles API Team

Delete API

  • user is member of the business group with roles API Team

  • note: an API cannot be deleted if it is referenced by API Products

Share with Business Groups

  • user can share with other business groups if they are member of the target business group with role API Team
