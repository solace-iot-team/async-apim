Installation

The installation starts all required services using this docker-compose-file:

qs-async-apim-www

an nginx service as a single point of entry, see nginx.conf for exposed services

qs-async-apim-admin-portal

the admin and developer portal web server

qs-async-apim-server

the portal server

qs-async-apim-server-mongodb

the portal database

qs-async-apim-connector

the connector

qs-async-apim-connector-mongodb

the connector database

  • Quickstart - quickstart directory in Repo.
